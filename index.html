📋 수집하는 정보

두잇노트는 앱 기능 제공을 위해 아래와 같이 기기 내에서만 데이터를 처리하며, 어떠한 개인 정보도 외부로 전송·저장하지 않습니다.

🔑 앱 권한 사용 목적

두잇노트가 요청하는 권한과 사용 목적은 다음과 같습니다.

권한 카메라 (CAMERA)
목적 사진 촬영 후 텍스트 인식(OCR) 기능 제공 — 촬영된 이미지는 기기 내에서만 처리되며 외부로 전송되지 않습니다.

권한 마이크 (RECORD_AUDIO)
목적 음성 메모 녹음 기능 — 녹음 데이터는 기기 내부에만 저장됩니다.

권한 알림 (POST_NOTIFICATIONS)
목적 리마인더 알림 발송 (Android 13 이상)

권한 정확한 알람 (SCHEDULE_EXACT_ALARM)
목적 사용자가 설정한 리마인더를 정확한 시간에 알림

권한 부팅 완료 수신 (RECEIVE_BOOT_COMPLETED)
목적 기기 재시작 후 리마인더 알람 자동 복원

권한 인터넷 (INTERNET)
목적 메모 내 외부 URL 이미지 미리보기 로딩 용도 — 개인 데이터는 전송하지 않습니다.

🔌 사용 중인 제3자 라이브러리

두잇노트는 아래 오픈소스 라이브러리를 사용하며, 모든 처리는 기기 내에서 이루어집니다.

  • ML Kit Text Recognition (Google) – OCR 텍스트 인식, 기기 온디바이스 처리
  • Room (Google AndroidX) – 로컬 데이터베이스
  • Coil – 이미지 로딩 라이브러리
  • Hilt (Google) – 의존성 주입
위 라이브러리는 사용자의 개인 데이터를 외부 서버로 전송하거나 광고·분석 목적으로 활용하지 않습니다.

🗂️ 데이터 보관 및 삭제

  • 모든 메모 및 설정 데이터는 사용자의 기기 내부 저장소에만 보관됩니다.
  • 앱을 삭제하면 모든 데이터가 영구적으로 삭제됩니다.
  • 앱 내 백업/복원 기능을 통해 사용자가 직접 데이터를 관리할 수 있습니다.

👶 아동 개인정보 보호

두잇노트는 만 14세 미만 아동으로부터 의도적으로 개인 정보를 수집하지 않습니다. 아동 사용자의 경우 보호자의 지도하에 앱을 사용하도록 권장합니다.
