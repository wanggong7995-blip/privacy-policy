# 🌱 마음돌봄 — 30일 자존감 회복 워크북

브라우저에서 바로 쓰는 자존감 회복 일기/워크북입니다. 서버·회원가입 없이, 모든 기록은 사용자의 기기(브라우저 localStorage)에만 저장됩니다.

## 시작하기

`self-esteem.html` 파일을 브라우저로 열기만 하면 됩니다. (더블클릭 또는 브라우저에 드래그)

## 구성

30일 프로그램으로, 매일 아래 흐름을 따라 5분 정도 기록합니다.

| 단계 | 내용 |
| --- | --- |
| 오늘의 확언 | 날마다 바뀌는 자기 긍정 문장 (30개) |
| 오늘의 마음 | 지금 자존감을 1~10으로 체크인 |
| 오늘의 질문 | 자기이해를 돕는 성찰 질문 (30개) |
| 작은 성취 | 사소해도 좋은 오늘의 성취 3가지 |
| 나에게 건네는 말 | 스스로에게 하는 칭찬 + 감사한 것 |

- **진행 현황**: 완료한 날, 연속 기록(스트릭), 30일 진행률을 보여줍니다.
- **지난 기록**: 날짜별로 펼쳐 다시 볼 수 있습니다.
- **내보내기**: 전체 기록을 텍스트 파일로 저장할 수 있습니다.

## 홈 화면에 추가 (앱처럼 사용)

이 워크북은 PWA(웹앱)라서 스마트폰 홈 화면에 아이콘으로 추가해 앱처럼 쓸 수 있고, 한 번 연 뒤에는 오프라인에서도 열립니다.

- **Android (Chrome)**: 접속하면 상단에 "추가" 안내가 뜹니다. 또는 메뉴(⋮) → "홈 화면에 추가".
- **iPhone (Safari)**: 하단 **공유** 버튼 → **"홈 화면에 추가"**.

## 개인정보

- 어떤 데이터도 외부로 전송되지 않습니다. (인터넷 연결 없이도 동작)
- 기록을 지우려면 앱 내 "모든 기록 삭제" 또는 브라우저 데이터 삭제.
